# Fieldnotes Lite - 核心优势与差异化定位

**创建日期**: 2025-10-20  
**版本**: 0.1.1

---

## 🎯 核心定位

### 不是什么
- ❌ 不是 FieldWorks 的替代品（太复杂）
- ❌ 不是 ELAN 的替代品（专注音视频）
- ❌ 不是 Toolbox 的替代品（格式专用）

### 是什么
- ✅ **跨平台**的轻量级语料管理工具
- ✅ **专注导出**的数据整理工具
- ✅ **易于上手**的田野笔记助手
- ✅ **透明表格**的论文写作帮手

---

## 🏆 两大核心优势

### 优势 1: 跨平台 🌍

**现状**:
- ✅ Windows / macOS / Linux 全支持
- ✅ 统一的用户体验
- ✅ 数据库文件完全兼容
- ✅ 无需云服务，本地运行

**竞品对比**:
| 工具 | Windows | macOS | Linux | 跨平台数据 |
|------|---------|-------|-------|-----------|
| **Fieldnotes** | ✅ | ✅ | ✅ | ✅ |
| FieldWorks | ✅ | ⚠️ | ⚠️ | 部分 |
| ELAN | ✅ | ✅ | ✅ | ✅ |
| Toolbox | ✅ | ❌ | ⚠️ | ✅ |

**差异化价值**:
> "在任何电脑上都能工作，数据随时可用"

### 优势 2: 智能导出对齐语料 📄

**核心特性**:
- ✅ Word 透明表格格式
- ✅ 词对词自动对齐
- ✅ 三行标准格式（原文/注释/翻译）
- ✅ 符合学术规范
- ✅ 可直接用于论文写作

**导出示例**:
```
┌────────────────────────────────┐
│ ŋa˧ tə˥ tɕʰi˥ fan˨˩            │  ← 原文（自动对齐）
├────────────────────────────────┤
│ 1SG CLF eat rice               │  ← 词汇分解（词对词）
├────────────────────────────────┤
│ 我吃饭                          │  ← 翻译
└────────────────────────────────┘
```

**竞品对比**:
| 工具 | 导出Word | 自动对齐 | 透明表格 | 可编辑 |
|------|---------|---------|---------|--------|
| **Fieldnotes** | ✅ | ✅ | ✅ | ✅ |
| FieldWorks | ⚠️ | ✅ | ❌ | ⚠️ |
| ELAN | ❌ | ❌ | ❌ | ❌ |
| Toolbox | ⚠️ | ⚠️ | ❌ | ⚠️ |

**差异化价值**:
> "从田野数据到论文，一键完成"

```

┌─────────────────────────────────┐
│ 样式模板:                        │
│ ○ 标准三行（当前）              │
│ ○ 四行格式（+音标层）            │
│ ○ 紧凑格式（节省空间）          │
│ ○ 语言学期刊格式                │
│ ○ 学位论文格式                  │
│                                 │
│ 自定义选项：                    │
│ ☑ 显示例句编号                  │
│ ☑ 首行缩进                      │
│ ☑ 表格边框（透明/细线/粗线）    │
│ ☑ 添加标题行                    │
│ □ 分页符（每N条语料）            │
│                                 │
│ 字体设置：                      │
│ 原文: [Charis SIL] [12pt]      │
│ 注释: [Courier New] [10pt]      │

│                                 │
│ [保存为模板] [应用] [重置]      │
└─────────────────────────────────┘
```

**预计工时**: 3-4天

#### 1.2 批量导出增强（高优先级）
**价值**: 提高大规模数据处理效率

**功能**:
- 按标签分章节导出
- 自动生成目录
- 添加页眉页脚（项目信息）
- 生成索引表
- 符号对照表

**导出结构**:
```
论文.docx
├─ 封面
├─ 目录（自动）
├─ 第一章 基本句型
│   ├─ 1.1 陈述句（标签过滤）
│   ├─ 1.2 疑问句
│   └─ 1.3 祈使句
├─ 第二章 日常对话
│   ├─ 2.1 问候
│   └─ 2.2 饮食
├─ 附录
│   ├─ 符号对照表
│   └─ 例句索引
```

**预计工时**: 4-5天

#### 1.3 多格式导出（中优先级）
**价值**: 扩展使用场景

**支持格式**:
1. **LaTeX** (gb4e/linguex) - 用于学术出版
   ```latex
   \begin{exe}
   \ex 
   \gll ŋa˧ tə˥ tɕʰi˥ fan˨˩\\
        1SG CLF eat rice\\
   \glt `我吃饭'
   \end{exe}
   ```

2. **HTML** - 用于网站展示
   ```html
   <div class="gloss">
     <div class="line1">ŋa˧ tə˥ tɕʰi˥ fan˨˩</div>
     <div class="line2">1SG CLF eat rice</div>
     <div class="line3">我吃饭</div>
   </div>
   ```

3. **Markdown** - 用于文档分享
   ```markdown
   | ŋa˧ | tə˥ | tɕʰi˥ | fan˨˩ |
   |-----|-----|-------|-------|
   | 1SG | CLF | eat   | rice  |
   
   '我吃饭'
   ```

4. **纯文本** - 用于邮件/即时通讯
   ```
   ŋa˧    tə˥    tɕʰi˥  fan˨˩
   1SG    CLF    eat    rice
   '我吃饭'
   ```

**预计工时**: 每个格式 2-3天

#### 1.4 导出预览（中优先级）
**价值**: 所见即所得，减少反复调整

**功能**:
```
┌─────────────────────────────────┐

│ │                           │  │
```

## 💡 核心原则

### 1. 专注核心优势
```
✅ 做什么:
   - 跨平台运行
   - 完美的 Word 导出
   - 简单易用
   - 快速高效

❌ 不做什么:
   - 复杂的词典管理（留给 FLEx）
   - 音视频标注（留给 ELAN）
   - 复杂的形态分析
```

### 2. 保持轻量
```
目标:
- 启动时间 < 3秒
- 内存占用 < 200MB
- 安装包 < 100MB
- 学习时间 < 5分钟
```

### 3. 用户至上
```
决策依据:
1. 能让用户更快完成工作吗？
2. 会增加学习成本吗？
3. 跨平台都能正常工作吗？
4. 导出结果更好了吗？
```
