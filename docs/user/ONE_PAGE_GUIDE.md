# Fieldnotes Lite - 一页纸快速指南

**🎯 目标用户**：没有编程经验的语言学研究者

## 📥 如何获取软件？

### 方法 1：下载可执行文件（推荐！）

**就像下载微信、QQ一样简单！**

1. 访问：https://github.com/yourusername/fieldnote/releases
2. 下载对应版本：
   - Windows：`Fieldnotes-Setup.exe`
   - Mac：`Fieldnotes.dmg`
   - Linux：`Fieldnotes-linux.tar.gz`
3. 双击安装，完成！

### 方法 2：找同事要安装包

让已经在用的同事分享给您，通过U盘或网盘传输。

### 方法 3：Python 安装

需要先安装 Python，详见《非技术用户使用指南》。

## 🚀 基本操作（5分钟学会）

### 1. 启动程序

双击桌面图标或程序文件。

### 2. 录入语料

| 字段 | 说明 | 示例 |
|------|------|------|
| 例句编号 | 可选，自定义编号 | CJ001 |
| 原文 | 待分析的语料文本 | ŋa˧ tə˥ tɕʰi˥ fan˨˩ |
| 词汇分解 | 形态学分析 | 1SG CLF eat rice |
| 翻译 | 自由翻译 | 我吃饭 |
| 备注 | 可选，附加说明 | 日常用语 |

点击"添加语料"保存。

### 3. 查找语料

切换到"检索"标签 → 输入关键词 → 点击"搜索"

### 4. 导出文档

切换到"导出"标签 → 点击"导出到 Word" → 选择保存位置

## 💾 重要：数据备份

**您的数据在哪里？**  
程序文件夹中的 `corpus.db` 文件

**如何备份？**  
复制 `corpus.db` 到 U盘/云盘/其他位置

**多久备份一次？**  
建议每周一次，或每次录入大量数据后

## ❓ 三个最常见问题

### Q: 需要联网吗？
**A:** 不需要！完全离线使用。

### Q: 数据会丢吗？
**A:** 不会。但请定期备份 `corpus.db` 文件。

### Q: 支持 IPA 音标吗？
**A:** 完全支持！直接粘贴即可。

## 🆘 遇到问题？

1. **程序打不开**：检查任务栏是否已经在运行
2. **提示"程序已在运行"**：关闭已有窗口，或重启电脑
3. **找不到数据文件**：查看程序文件夹中的 `corpus.db`
4. **其他问题**：查看《非技术用户使用指南》或联系提供者
