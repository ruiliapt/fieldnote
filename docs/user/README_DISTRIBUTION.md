# 面向非技术用户的发布建议

## 📦 为语言学研究者准备软件包

如果您想让同事（没有编程经验）使用 Fieldnotes Lite，以下是最佳发布方式：

---

## ⭐ 推荐方案：提供可执行文件

### 为什么？
- ✅ **无需安装 Python**
- ✅ **双击即用**，就像普通软件
- ✅ **适合所有用户**，包括完全不懂电脑的

### 如何制作？

#### 1. 构建可执行文件

```bash
# macOS/Linux
./build_executable.sh

# Windows
build_executable.bat
```

生成的文件在 `dist/` 目录：
- **macOS**: `Fieldnotes.app`
- **Windows**: `Fieldnotes.exe`
- **Linux**: `Fieldnotes`

#### 2. 测试

在一台**没有安装 Python** 的电脑上测试，确保能正常运行。

#### 3. 打包分发

**方式 1: 创建安装包**
- macOS: 制作 DMG 文件
- Windows: 使用 Inno Setup 制作 Setup.exe
- Linux: 制作 .tar.gz 或 .deb/.rpm

**方式 2: 简单压缩包**
- 把 `dist/Fieldnotes/` 整个文件夹压缩
- 添加使用说明（见下方）
- 通过网盘/U盘分发

#### 4. 准备使用说明

在压缩包中包含：
- `使用说明.txt` - 简单的文字说明
- `一页纸快速指南.pdf` - 从 ONE_PAGE_GUIDE.md 导出
- `示例数据.csv` - 几条示例语料

## 📄 使用说明示例

创建 `使用说明.txt`：

```
Fieldnotes Lite - 田野笔记管理工具
==================================

📥 安装方法
----------
Windows 用户：
  1. 解压文件到任意位置（如"文档"文件夹）
  2. 双击 Fieldnotes.exe 启动程序
  3. 完成！

Mac 用户：
  1. 解压文件
  2. 双击 Fieldnotes.app
  3. 如果提示"无法验证开发者"：
     右键点击 → 选择"打开" → 点击"打开"
  4. 完成！

Linux 用户：
  1. 解压文件
  2. 打开终端，cd 到解压目录
  3. 运行：./Fieldnotes
  4. 完成！

🚀 基本使用
----------
1. 启动程序后，在"数据管理"标签页输入数据
2. 点击"添加语料"保存
3. 在"检索"标签页可以搜索
4. 在"导出"标签页可以导出到 Word

💾 数据备份
----------
您的数据保存在：程序文件夹/corpus.db
建议定期复制此文件备份！

❓ 常见问题
----------
Q: 需要联网吗？
A: 不需要，完全离线使用。

Q: 数据会丢吗？
A: 不会，但请定期备份 corpus.db 文件。

Q: 支持 IPA 音标吗？
A: 完全支持，直接粘贴即可。

📚 详细教程
----------
更多信息请查看：
https://github.com/yourusername/fieldnote

或联系：your.email@example.com
```

## 🎓 面向机构/实验室的部署方案

### 方案 1: 网络共享文件夹

1. **在服务器上设置共享文件夹**
2. **所有人安装可执行文件到本地**
3. **数据库文件放在共享文件夹**

⚠️ **注意**：不要多人同时编辑数据库！建议：
- 分配不同的数据库文件
- 定期合并数据

### 方案 2: 云盘同步

1. **每人独立使用**
2. **数据库文件放在云盘同步文件夹**
3. **定期导出备份**

⚠️ **注意**：避免同时编辑导致冲突。

### 方案 3: 集中培训

1. **组织一次集体培训**（30分钟）
   - 演示安装
   - 演示基本操作
   - 解答疑问

2. **指定技术支持人员**
   - 处理安装问题
   - 数据备份和恢复
   - 定期整理数据

3. **建立使用群组**
   - 微信群/钉钉群
   - 分享使用技巧
   - 互相帮助
