**专为语言学研究者设计的田野笔记管理工具**

本指南面向**没有编程经验**的用户，用最简单的方式说明如何安装和使用。

## 📥 方式一：下载可执行文件（最简单，推荐！）

### ✨ 适合：完全不想碰代码的用户

**这是最简单的方式，就像安装微信、QQ一样！**

### 下载步骤

1. **访问下载页面**
   - 打开浏览器
   - 访问：`https://github.com/yourusername/fieldnote/releases`
   - （将 `yourusername` 替换为实际用户名）

2. **选择您的系统**
   - **Windows 用户**：下载 `Fieldnotes-Setup-0.1.0.exe`
   - **Mac 用户**：下载 `Fieldnotes-0.1.0.dmg`
   - **Linux 用户**：下载 `Fieldnotes-0.1.0-linux.tar.gz`

3. **安装**
   - **Windows**：双击 `.exe` 文件，按提示安装
   - **Mac**：双击 `.dmg` 文件，拖动到"应用程序"文件夹
   - **Linux**：解压后运行 `Fieldnotes` 文件

4. **运行程序**
   - 在"开始菜单"（Windows）或"启动台"（Mac）中找到 Fieldnotes
   - 双击图标启动

> ✅ **完成！** 不需要安装 Python，不需要命令行，就像使用普通软件一样！

## 📥 方式三：Python 安装（需要一点学习）

### ✨ 适合：愿意学一点电脑操作的用户

这种方式需要安装 Python，但我们会**一步步指导**您。

### 🪟 Windows 用户

#### 第 1 步：安装 Python

1. **下载 Python**
   - 访问：https://www.python.org/downloads/
   - 点击黄色大按钮"Download Python"
   
2. **安装 Python**
   - 双击下载的文件
   - ⚠️ **重要**：勾选"Add Python to PATH"（在底部）
   - 点击"Install Now"
   - 等待安装完成

3. **验证安装**
   - 按 `Win + R` 键
   - 输入 `cmd`，按回车
   - 在黑色窗口中输入：`python --version`
   - 如果显示版本号，说明安装成功

#### 第 2 步：下载 Fieldnotes

1. **下载项目文件**
   - 访问：https://github.com/yourusername/fieldnote
   - 点击绿色按钮"Code"
   - 选择"Download ZIP"
   - 解压到您容易找到的位置（如"文档"文件夹）

#### 第 3 步：安装依赖

1. **打开命令提示符**
   - 在解压的文件夹中，按住 `Shift` 键
   - 右键点击空白处
   - 选择"在此处打开 PowerShell 窗口"

2. **安装必要的软件包**
   - 复制这行命令：`pip install -r requirements.txt`
   - 粘贴到窗口中，按回车
   - 等待安装完成（可能需要几分钟）

#### 第 4 步：运行程序

1. **启动 Fieldnotes**
   - 双击 `run.bat` 文件
   - 程序窗口会自动打开

2. **创建快捷方式**（可选）
   - 右键点击 `run.bat`
   - 选择"发送到" → "桌面快捷方式"
   - 以后双击桌面图标即可启动

---

### 🍎 Mac 用户

#### 第 1 步：安装 Python

1. **检查是否已安装**
   - 打开"启动台"
   - 搜索"终端"（Terminal）
   - 输入：`python3 --version`
   - 如果显示版本号 ≥ 3.11，跳到第 2 步

2. **安装 Python**（如果需要）
   - 访问：https://www.python.org/downloads/
   - 下载 Mac 版本
   - 双击安装包，按提示安装

#### 第 2 步：下载 Fieldnotes

1. **下载项目文件**
   - 访问：https://github.com/yourusername/fieldnote
   - 点击绿色按钮"Code"
   - 选择"Download ZIP"
   - 解压（双击 ZIP 文件）

#### 第 3 步：安装依赖

1. **打开终端**
   - 启动台 → 终端
   - 输入：`cd ` （cd后面有个空格）
   - 把解压的文件夹拖到终端窗口
   - 按回车

2. **安装软件包**
   - 复制：`pip3 install -r requirements.txt`
   - 粘贴到终端，按回车
   - 输入电脑密码（看不见是正常的）
   - 等待安装完成

#### 第 4 步：运行程序

1. **启动 Fieldnotes**
   - 在终端中输入：`./run.sh`
   - 按回车
   - 程序窗口会打开

2. **添加到 Dock**（可选）
   - 程序运行后，在 Dock 中右键点击图标
   - 选择"选项" → "在 Dock 中保留"

## 📖 开始使用

### 首次打开程序

启动后，您会看到三个标签页：

1. **数据管理** - 录入和管理您的语料
2. **检索** - 搜索已录入的数据
3. **导出** - 导出为 Word 文档

### 基本操作流程

#### 1️⃣ 录入数据

在"数据管理"标签页：

```
例句编号：CJ001 （可选，如"藏语001"）
原文：ŋa˧ tə˥ tɕʰi˥ fan˨˩
词汇分解：1SG CLF eat rice
翻译：我吃饭
备注：日常用语 （可选）
```

点击"添加语料"保存。

#### 2️⃣ 编辑数据

- 点击表格中的某一行
- 数据会自动填充到输入框
- 修改后点击"更新语料"

#### 3️⃣ 删除数据

- 点击要删除的行
- 点击"删除语料"
- 确认删除

#### 4️⃣ 搜索数据

切换到"检索"标签页：
- 选择搜索字段（或"全部字段"）
- 输入关键词
- 点击"搜索"

#### 5️⃣ 导出 Word 文档

切换到"导出"标签页：
1. 设置参数（使用默认值即可）
2. 选择导出范围（全部或搜索结果）
3. 点击"导出到 Word"
4. 选择保存位置
5. 完成！

## 💾 数据备份

### 您的数据在哪里？

所有数据保存在：`程序文件夹/corpus.db`

### 如何备份？

**最简单的方法**：
1. 找到程序文件夹
2. 复制 `corpus.db` 文件
3. 保存到 U盘/云盘/其他位置

**推荐备份频率**：每周一次，或每次录入大量数据后

### 如何恢复？

把备份的 `corpus.db` 文件复制回程序文件夹，替换原文件即可。

## ❓ 常见问题（非技术版）

### Q1: 我完全不懂电脑，能用吗？

**答**：可以！如果有同事已经安装好，您只需要：
1. 双击桌面图标
2. 在表格中输入数据
3. 点击"添加"按钮
就这么简单！

### Q2: 需要联网吗？

**答**：不需要！这是离线软件，您的数据都存在自己的电脑上，非常安全。

### Q3: 数据会丢吗？

**答**：不会。但建议定期复制 `corpus.db` 文件作为备份。

### Q4: 可以在多台电脑上使用吗？

**答**：可以！复制整个程序文件夹到其他电脑即可。或者只复制 `corpus.db` 文件。

### Q5: 支持 IPA 音标吗？

**答**：完全支持！直接粘贴或使用 IPA 输入法输入即可。

### Q6: 导出的 Word 文档可以编辑吗？

**答**：可以！导出后就是普通的 Word 文件，可以随意编辑。

### Q7: 可以导入我已有的数据吗？

**答**：可以！把数据整理成 Excel 表格，另存为 CSV 格式，然后点击"批量导入"。

### Q8: 程序打不开怎么办？

**答**：
1. 检查是否已经有一个窗口在运行（看任务栏）
2. 重启电脑试试
3. 重新下载安装
4. 联系提供者寻求帮助

### Q9: 我的数据在哪里？

**答**：在程序文件夹里，有个叫 `corpus.db` 的文件，那就是您的全部数据。

### Q10: 可以多人同时使用吗？

**答**：不建议。每台电脑独立使用，然后通过 U盘或网盘交换 `corpus.db` 文件。

## 💡 使用技巧

### 技巧 1：快速录入

使用 `Tab` 键在输入框间切换，不用鼠标点击。

### 技巧 2：复制粘贴

从 Word/Excel 复制文本，直接粘贴到输入框。

### 技巧 3：模糊搜索

搜索时不需要输入完整词语，输入部分即可。

### 技巧 4：批量处理

如果有大量数据，先在 Excel 中整理好，再批量导入。

### 技巧 5：定期备份

每次工作结束前，复制一份 `corpus.db` 到云盘。

## 📚 推荐工作流程

### 田野调查中

1. **现场记录**：在笔记本/录音笔上记录
2. **当天整理**：回到住处后录入 Fieldnotes
3. **每日备份**：复制数据库文件到云盘
4. **每周导出**：导出 Word 文档查看整体进度

### 论文写作时

1. **检索相关例句**：使用搜索功能
2. **导出到 Word**：只导出需要的例句
3. **在 Word 中编辑**：调整格式，添加序号
4. **插入论文**：复制粘贴到论文中
